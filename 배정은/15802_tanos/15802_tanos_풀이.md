# 백준 15802

#  풀이

### 문제

이 일이 끝나면 인류의 절반은 살아남을 것이다. 

---------

### 입력

이 문제는 입력이 주어지지 않는다. 

------

### 출력

당신이 살아남는다면 1을, 살아남지 못한다면 0을 출력한다. 

------

### 풀이

* 0 또는 1을 출력하였을때, '맞았습니다!!'가 뜨는 경우가 맞는 답
* 나의 경우에는, 0을 출력한 경우에는 '틀렸습니다'였고, 1을 출력한 경우에는 '맞았습니다!!'였음
* 즉, 나는 타노스로 부터 살아남았음

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
	printf("1\n"); //나는 살아 남는다!!
	return 0;
}
```
